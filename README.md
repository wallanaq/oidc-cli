`oidc-cli` is a command-line tool written in Go that provides utilities for working with **OpenID Connect (OIDC)** and **OAuth 2.0**. It allows developers and operators to interact with identity providers for discovery, token generation, validation, introspection, and JWT decoding — all from the terminal.

### ✨ Key Features

- Fetch and inspect OIDC discovery documents
- Generate access tokens via multiple OAuth2 flows
- Decode and verify JWTs
- Introspect tokens with the OIDC provider
- Validate tokens locally or remotely
- CLI-based automation and scripting support

### 🧪 Project Status

This project is in **early development** (v0.1.0). Features and commands will evolve rapidly.
